# Skill: 笔记管理

## 描述
帮助用户在工作区创建、查找和管理笔记。

## 数据目录
`workspace/notes/`

## 操作

### 创建笔记
当用户说"帮我记一下"、"写个笔记"、"保存这段话"等：
1. 提取笔记内容和可选标题
2. 如果没有标题，从内容生成简短标题
3. 创建文件 `workspace/notes/<日期>-<标题>.md`
4. 写入内容（Markdown 格式，顶部加日期时间）
5. 确认保存

### 查找笔记
当用户说"找笔记"、"之前记的"、"搜索笔记"等：
1. 列出 `workspace/notes/` 目录
2. 根据关键词匹配文件名或内容
3. 展示匹配的笔记

### 列出笔记
当用户说"所有笔记"、"笔记列表"：
1. 列出 `workspace/notes/` 目录所有文件
2. 按时间排序显示

### 删除笔记
当用户确认要删除某个笔记时：
1. 找到对应文件
2. 删除并确认
