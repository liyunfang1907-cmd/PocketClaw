# Skill: 待办清单管理

## 描述
帮助用户管理待办事项列表。可以添加、完成、删除和查看待办。

## 数据文件
`workspace/todo.json`

## 操作

### 添加待办
当用户说"添加待办"、"提醒我"、"记一下"等：
1. 读取 `workspace/todo.json`（不存在则创建空数组）
2. 添加新条目：`{"id": <时间戳>, "text": "<内容>", "done": false, "created": "<时间>"}`
3. 保存文件
4. 回复确认

### 查看待办
当用户说"看看待办"、"还有什么没做"等：
1. 读取 `workspace/todo.json`
2. 按状态分组显示（未完成在前）
3. 如果为空，告知用户

### 完成待办
当用户说"完成了"、"做完了"加上具体内容：
1. 读取文件，找到匹配条目
2. 设置 `done: true`，添加 `completedAt`
3. 保存并确认

### 删除待办
当用户说"删除"、"去掉"加上具体内容：
1. 从列表中移除匹配条目
2. 保存并确认
