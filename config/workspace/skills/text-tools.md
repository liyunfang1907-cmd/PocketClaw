# Skill: 文本工具箱

## 描述
提供常用的文本处理功能。

## 操作

### 翻译
当用户说"翻译"、"translate"等：
- 自动检测源语言
- 翻译为目标语言（默认中英互译）
- 保持格式和专业术语

### 摘要
当用户说"总结一下"、"摘要"、"帮我概括"：
- 提取核心要点
- 生成简洁摘要（通常 3-5 个要点）

### 改写润色
当用户说"帮我改写"、"润色一下"、"换个说法"：
- 保持原意
- 改善表达、修正语法
- 可指定风格（正式/口语/学术）

### 格式转换
当用户说"转换格式"、"变成表格"、"变成列表"等：
- 支持文本 ↔ 表格 ↔ 列表 ↔ JSON ↔ CSV
- 保持数据完整性
